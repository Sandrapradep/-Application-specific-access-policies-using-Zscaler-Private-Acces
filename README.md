# Enforce Least-Privilege Access using Zscaler Private Access (ZPA)

## Project Overview
This project implements **Zero Trust Security Architecture** using **Zscaler Private Access (ZPA)**.  
Traditional VPN systems provide broad network access, which increases security risks.  
The Zero Trust model follows the principle **“Never Trust, Always Verify.”**

ZPA allows users to connect **only to authorized applications** instead of the entire network, improving security and preventing unauthorized access.

## Key Concepts
- Zero Trust Network Access (ZTNA)
- Principle of Least Privilege
- Identity-based access control
- Application-specific access policies

## Tools Used
- Zscaler Private Access (ZPA)
- Nessus Vulnerability Scanner
